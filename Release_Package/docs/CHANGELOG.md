# Changelog

## 0.1.0

- Переход на Single-window UI: весь прогресс (сканирование, копирование, ETA, скорость) отображается в нижней панели главного окна без всплывающих окон.
- Добавлена отмена операций (сканирование/копирование) через кнопку в нижней панели.
- Исправлена ошибка Tkinter callback при отмене передачи (`NameError` в `on_err`).
- Исправлено case-sensitive сравнение файлов: файлы с именами разного регистра корректно считаются идентичными (актуально для Windows).
- Зафиксированы версии основных зависимостей в `source/requirements.txt`.
- Известное ограничение: при многократной быстрой отмене подряд возможны единичные `Permission denied` для отдельных файлов в синхронизируемых каталогах (OneDrive и др.). В обычном сценарии не воспроизводится стабильно.
